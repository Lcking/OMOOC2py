# github 私人教程

## 背景
- 体验程序员的既视感
- 熟悉远程协同办公的环境
- 想教会半年前的自己甚至是半年前的你

## 安装
- github.com注册账号
- 记住账号密码

## 配置
- 无需过多配置,到手即用


## 使用
- 创建repository
- 填写名称，填写描述
- 选择公开-create repository,这样一个公开的仓库就已经建立好了！
